Welcome to the Housing API

This API provides data about housing prices and demographics in California.

API Endpoints

  • /api/housing: returns all housing data
  • /api/housing/distance_to_la/50: returns all housing data within 50 miles of Los Angeles
  • /api/housing/distance_to_sf/50: returns all housing data within 50 miles of San Francisco
  • /api/housing/distance_to_sd/50: returns all housing data within 50 miles of San Diego
  • /api/housing/distance_to_sj/50: returns all housing data within 50 miles of San Jose
  • /api/housing/distance_to_coast/10: returns all housing data within 10 miles of the coast
  • /api/housing/median_house_value/500000: returns all housing data with a median house value of $500,000
  • /api/housing/median_income/50000: returns all housing data with a median income of $50,000
  • /api/housing/median_age/30: returns all housing data with a median age of 30
  • /api/housing/tot_rooms/4: returns all housing data with 4 total rooms
  • /api/housing/tot_bedrooms/2: returns all housing data with 2 total bedrooms
  • /api/housing/population/1000: returns all housing data with a population of 1000
